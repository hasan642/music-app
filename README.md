# Musician
### Main Folder Structure
```bash
        ├── fake-json-server
        ├── .env
        ├── .env.production
        ├── App.tsx
        ├── src
        │   ├── api
        │   ├── assets
        │   ├── components
        │   ├── config
        │   ├── navigation
        │   ├── screens
        │   ├── theme
        │   └── utils
```
### Installation
```sh
$ git clone https://github.com/hasanal1995/music-app.git
$ cd music-app
$ yarn
$ cd ios && pod install
```
### Running for Android
```sh
$ yarn android
```
### Running for iOS
```sh
$ yarn ios
```

### Testing
```sh
$ yarn test
```
